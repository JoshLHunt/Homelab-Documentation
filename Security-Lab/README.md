# Security Lab Documentation

This section documents my security-focused lab environments used for offensive and defensive testing.

All environments are isolated, private, and sanitized for documentation purposes.

---

## Active Directory Lab

### Environment Overview
- Windows Server Domain Controller
- Domain-joined Windows client
- Kali Linux attacker machine
- Segmented lab network

### Objectives
- Practice enumeration and privilege escalation
- Simulate real-world attack chains
- Document findings and mitigations
